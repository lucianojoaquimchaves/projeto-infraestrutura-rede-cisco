# Projeto de Infraestrutura de Rede Cisco

## Descrição

Projeto académico desenvolvido no âmbito da disciplina de Redes de Computadores.

O objetivo foi criar e configurar uma infraestrutura de rede empresarial utilizando equipamentos Cisco em ambiente de simulação.

Foram aplicados conceitos de segmentação de rede, endereçamento IP, serviços de rede e comunicação entre diferentes segmentos.

## Tecnologias e ferramentas utilizadas

- Cisco Packet Tracer
- GNS3
- EVE-NG
- PnetLab
- Equipamentos Cisco (Routers e Switches)

## Configurações implementadas

- Configuração de VLANs
- Switching e Trunking
- Endereçamento IPv4 e IPv6
- DHCP
- DNS (quando aplicável)
- NAT/PAT
- ACLs
- Roteamento estático
- Protocolos de roteamento:
  - RIP
  - EIGRP

## Competências desenvolvidas

- Planeamento de topologias de rede
- Configuração de routers e switches Cisco
- Diagnóstico de problemas de conectividade
- Segmentação de redes através de VLANs
- Aplicação de boas práticas de redes empresariais

## Ficheiros

- Topologia da rede
- Configurações dos equipamentos
- Documentação do projeto
- Capturas de testes de conectividade
